# dida-util

一些常用的工具类和工具函数。它是 [宙品科技](http://zeupin.com) 开源的 [Dida框架](http://dida.zeupin.com) 的一个功能组件。

## 使用

用Composer安装：

```bash
composer require dida/util
```
